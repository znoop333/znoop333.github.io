CChart::GetMinorGridLineStyle

int GetMinorGridLineStyle( void );
void SetMinorGridLineStyle( int lineStyle );

Description

These functions set or retrieve the line style used for drawing the minor grid lines. They use the standard Windows line styles, PS_SOLID, PS_DASH, PS_DOT, PS_DASHDOT, PS_DASHDOTDOT.
