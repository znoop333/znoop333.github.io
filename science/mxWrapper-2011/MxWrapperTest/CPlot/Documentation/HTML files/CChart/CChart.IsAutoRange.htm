CChart::IsAutoRange

CChart::IsXAutoRange

CChart::IsYAutoRange

void IsAutoRange( BOOL &haveXRange, BOOL &haveYRange );
BOOL IsXAutoRange( void );
BOOL IsYAutoRange( void );

Parameters

  • haveXRange - Receives whether the X range has been set
  • haveYRange - Receives whether the Y range has been set

Description

These functions determine whether the X and Y directions are auto ranging based on what axes are present. By default, CChart and the derived functions assume that the bottom axis will determine the X range and the top axis will determine the Y range. However, in the case that a bottom axis doesn't exist, the top axis will determine the range. These functions will return the auto range flag for whatever axis controls the x or y range.
