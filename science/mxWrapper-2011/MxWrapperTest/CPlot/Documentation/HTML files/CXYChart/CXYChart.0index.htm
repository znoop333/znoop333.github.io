CXYChart member functions

These are the member functions of CXYChart:

Construction/Destruction

  • CXYChart - Default constructor
  • ~CXYChart - Default destructor
Overridden
  • OnDraw - Draw the chart
  • Serialize - Save or restore a chart
  • WriteTextFile - Save the data to a text file
  • GetPlotRect - Return the rect of the data region of the plot
  • DeleteAllData - Delete every data set
  • DeleteData - Delete a specific data set
  • AxisSettings - Show the axis settings dialog box
  • DataSettings - Show the data settings dialog box
  • PlotSettings - Show the plot settings dialog box
  • ShowPopupMenu - Show the shortcut menu for the chart
  • AddData - Add a data set to the chart
  • SetData - Change the data in an existing data set
Data set properties
  • GetDataColor - Get the color used to draw the data set
  • SetDataColor - Set the color used to draw the data set
  • GetMarkerType - Get the color used to draw the data set
  • SetMarkerType - Set the marker type used to draw the data set
  • GetChartType - Get the chart type (scatter, line, etc) used to draw the data set
  • SetChartType - Set the chart type (scatter, line, etc) used to draw the data set
  • GetMarkerFrequency - Get the frequency of the markers
  • SetMarkerFrequency - Set the frequency of the markers
  • GetMarkerFillState - Get the state of filling of the markers
  • SetMarkerFillState - Set the state of filling of the markers
  • GetDataLineSize - Get the line size used to draw the data set
  • SetDataLineSize - Set the line size used to draw the data set
  • GetDataLineStyle - Get the line style used to draw the data set
  • SetDataLineStyle - Set the line style used to draw the data set
Utility
  • CopyChartData - Copy drawing settings from one chart to another
  • GetRangeFromData - Get the range of a data set from the data
  • DrawDataSet - Draw the data sets
  • DrawMarker - Draw a marker at a place, if needed
Popup menu
  • InterpretPopupMenuItem - Interpret the selection of a popup menu
  • AddDataPopupMenus - Make a popup menu for data sets
  • AddChartTypePopup - Make a popup menu for chart types
  • AddLineSizePopup - Make a popup menu for line sizes
  • AddMarkerPopup - Make a popup menu for marker types and frequency
  • InterpretDataSetPopupMenuItem - Interpret the selection of an item from a data set menu.
Private
  • GetDataRanges - Get the ranges of the data from the data, and store it internally
  • GetDataRange - Get the range of one data set from the data, and store it internally
  • SetDefaults - Set the plot settings to the default
