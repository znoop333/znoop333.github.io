CXYChart::DeleteData

virtual BOOL DeleteData( int dataRef );

Parameters

  • dataRef - A data reference number returned by AddData or SetData

Returns

TRUE if the delete was sucessful, FALSE otherwise

Description

This function deletes a specific data set. Since all the settings are array based (instead of dynamic), this requires that the settings be moved around within the arrays to account for the newly missing member. This function does this for the plot color, style, markers, etc.
