CXYChart::GetMarkerType

CXYChart::SetMarkerType

void SetMarkerType( int dataID, int type );
int GetMarkerType( int dataID );

Description

These functions set or return the marker type used for a specific data set. Possible marker types are 

kXYMarkerNone
kXYMarkerCircle
kXYMarkerSquare
kXYMarkerTriangle
kXYMarkerX

The data ID is that returned by AddData or SetData.
