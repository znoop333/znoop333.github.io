CXYChart::InterpretDataSetPopupMenuItem

int InterpretDataSetPopupMenuItem( int whichDataSet, int result );

Parameters

  • whichDataSet - The index of the data set in question
  • result - The menu item selected (the menu item % kMenuDataRange)

Returns

Either a popup constant if the menu item was interpreted, or the menu item number if not interpreted.
