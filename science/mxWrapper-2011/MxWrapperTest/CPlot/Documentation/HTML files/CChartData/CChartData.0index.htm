CChartData member functions

CChartData is classed entirely separately from the charts so that it can serve multiple uses. It can be used as a data container for the charts or any other 1-3 dimensional data that you might care to store.

These are the member functions of CChartData:

Construction/Destruction

  • CChartData - Default constructor
  • ~CChartData - Default destructor
Overridable
  • Serialize - Save or restore data
Other
  • DeleteData - Delete a specific data set
  • GetData - Retrieve a pointer to the stored double data
  • SetData - Set the data values
  • SetDimensions - Change dimensions of the data
  • GetDimensions - Get the dimensions of the data
  • GetNDimensions - Get the dimension count of the data
