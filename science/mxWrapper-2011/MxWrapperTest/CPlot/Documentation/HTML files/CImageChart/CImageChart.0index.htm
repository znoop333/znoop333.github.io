CImageChart member functions

CImageChart is derived from CChart, and thus inherets all the functionality of the parent class. The functions listed here are specific to image charts or are overrides of the parent function.

CImageChart allows for only one data set at a time. Thus the data functions below will modify the current data set. Even the AddData function is overloaded to overwrite the current data.

CImageChart has specific ways of drawing images. See the variables page for a description of the three drawing modes. It also uses pre-made color tables for displaying the data. More information on the color tables can be found on the GetPalette page.

These are the member functions of CImageChart:

Construction/Destruction

  • CImageChart - Default constructor
  • ~CImageChart - Default destructor
Overridden
  • OnDraw - Draw the chart
  • Serialize - Save or restore a chart
  • GetPlotRect - Return the rect of the data region of the plot
  • DeleteData - Delete a specific data set
  • AxisSettings - Show the axis settings dialog box
  • PlotSettings - Show the plot settings dialog box
  • ShowPopupMenu - Show the shortcut menu for the chart
  • AddData - Change the current image
  • SetData - Change the data in an existing data set
Drawing
  • DrawPlotBasics - Erase the background and draw the frame
  • DrawDataSet - Draw the image
Data
  • SetUserByteImagePtr - Set a temporary pointer to a byte image to draw with
  • GetUserByteImagePtr - Retrieve the temporary pointer to a byte image to draw with
  • SetUserImagePtr - Set a temporary pointer to image data
  • SetUserImagePtr - Retrieve the temporary pointer to image data
  • GetScaledImageData - Create scaled image data
  • AddImage - Change the current image
  • DeleteImage -Delete the current image
Miscelaneous
  • GetPlotRange - Get the x and y range of the image
  • GetMinDataDimensions - Get the x and y size of the image
  • AdjustPlotSizeForWindowSize - Adjust the plot size to fit the window, depending on scaling options
  • GetPalette - Get the current palette for the image
  • SetPalette - Set the current palette for the image
  • SetPaletteMenu - Set the current palette for the image based of a menu selection
  • GetPlotScale - Get the current scaling factor for the image
  • InterpretPopupMenuItem - Interpret a user selection in a shortcut menu
Variables
  • m_PlotToWindowSize - Adjust the plot to the window size, or scale by factors of 2?
  • m_MaintainAspectRatio - Maintain the aspect ratio of the image when drawing
