CImageChart::SetUserByteImagePtr

CImageChart::GetUserByteImagePtr

void SetUserByteImagePtr( BYTE *imagePtr );
BYTE *GetUserByteImagePtr( void );

Parameters

  • imagePtr - A pointer to a word-aligned byte image of the same size as the current image

Description

This function allows the user to set a temporary byte image to draw from. This can be useful in situations where the data is changing frequently, or when the data is easier kept by the application than by CImageChart. In the case of a program I was writing, this was true since the data was modified frequently, and it would have been slow to keep calling SetData and having to copy the data over and over again.

Use this function sparingly, however, since it is best to store the image data with CImageChart. This way the serialization and other such things work.
