Tutorial

CPlot is relatively simple to add to an existing project. One simply copies the relavant source files into the project, adds some functions for creating the data set and the chart, and a function or two for drawing and printing.

This tutorial attempts to show each of these steps, to make adding CPlot to a project very simple.

  • Add the CPlot files and resources to your project
  • Add a chart object to your view
  • Create data for a chart
  • Create a chart and set up axes
  • Draw the chart
  • Print the graph
  • Add shortcut menu support
