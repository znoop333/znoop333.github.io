CImagePalette::GetCurPaletteNumber

CImagePalette::SetPalette

void SetPalette( int whichPalette );
int GetCurPaletteNumber( void );

Parameters

  • whichPalette - A palette index to change to

Description

This function sets or returns the current palette used by CImagePalette. The palettes are pre-created and stored in resources (see the tutorial for more information on CPlot resources), and loaded out of those resources for display. Experimentation is the best way to find the best palette.
