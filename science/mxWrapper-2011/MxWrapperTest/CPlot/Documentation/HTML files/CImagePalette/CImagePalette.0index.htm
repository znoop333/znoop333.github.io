CImagePalette member functions

CImagePalette is a simple class designed to manage pre-created color palettes and provide a bitmap info header for calls to such functions as StretchDIBits which requires one of those functions.

These are the member functions of CImagePalette:

Construction/Destruction

  • CImagePalette - Default constructor
  • ~CImagePalette - Default destructor
Other
  • GetCurPaletteNumber - Get the number of the current palette
  • SetPalette - Load another palette
  • GetBitmapInfo - Get a pointer to a BITMAPINFO structure
  • GetPalette - Get a pointer to the current CPalette
  • GetNumEntries - Get the number of available entries in the Windows color palette
