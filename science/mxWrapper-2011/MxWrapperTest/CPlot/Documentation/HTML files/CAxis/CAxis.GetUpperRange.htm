CAxis::GetUpperRange

double GetUpperRange( void );

Returns

A double representing the upper range of the axis. Note that this could be less than the lower range in the case of an axis plotted from high to low instead of low to high.

Description

If the axis is set for auto-ranging, it will return the auto-ranged value, provided the chart has been displayed. The value returned by GetUpperRange is invalid until the chart is displayed.
