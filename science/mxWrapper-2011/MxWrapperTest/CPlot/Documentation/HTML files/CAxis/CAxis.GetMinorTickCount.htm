CAxis::GetMinorTickCount

inline int GetMinorTickCount( void );
void SetMinorTickCount( int tickCount );

Description

These functions set or return the number ot minor tick marks along the current axis. For minor tick marks, this is strictly the number of tick marks (unlike for major tick marks where it is the number of intervals).
