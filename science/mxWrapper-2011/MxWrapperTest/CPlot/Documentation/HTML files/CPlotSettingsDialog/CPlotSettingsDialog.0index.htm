CPlotSettingsDialog members

CPlotSettingsDialog is derived from CDialog. It can be called simply from within CXYChart as follows: 
CPlotSettingsDialog		dialog( NULL, this );

if( dialog.DoModal() == IDOK )
{
	// Need to update chart
}
Although the class is public, it is better to call the Plot settings function of CXYChart directly.

CPlotSettingsDialog requires that the dialog resource with id IDD_XY_SETTINGS exist in the current file. See the tutorial for more information on copying resources.

These are the members of CPlotSettingsDialog:

Functions

  • CPlotSettingsDialog( CWnd* pParent, CXYChart *chart ) - The default constructor requires a pointer to an image chart object from which to obtain the current plot settings, and to copy the settings back into. The parent window can be NULL or point to the parent window.
